This is a demo of Electron.

You can use it to manage your servers.
It can execute customized comands on servers.

features:

* add server by ip, port, username, password
* add commands
* connect to a server and run the specified command

![alt text][settings] 
![alt text][server] 


## Run on local
```
yarn install
yarn start
```

## Build
```
yarn install
yarn dist
```
